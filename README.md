This repository contains course materials organized by module.

├── module2/          # Module 2 materials
│   ├── Module2-Figure2.2.png
│   ├── Module2-Figure2.4.png
│   ├── Module2-Figure2.5.png

    ├── stdn1.txt

## Folders

- **module1/**: Contains figures and video content for Module 1
- **module2/**: Contains figures and video content for Module 2
- **module3/**: Contains video content for Module 3
- **data/**: Contains text data files used in exercises and examples
