Looping Syntax#

Loops are used for repetitive tasks:

Instead of this:

print("string")  print("string")  print("string")  print("string") …

You can do this:

x=1  while x<5:     print("string")

x=1
while x<5:
    print("string")
    print("Current x  is :",x)
    x=x+1
    print("Final x  is :",x)

# find the maximum number below 1000 that divides by 12
for n in range(1000, 0, -1):
    print(n)
    div = n % 12
    print(div)
